Welcome to the **My Ankle Website** project, developed by **Biffnewmedia**. This project is built using the [T3 Stack](https://create.t3.gg/) and serves as the official website for My Ankle. It leverages the power of [Prismic](https://prismic.io) as its Content Management System (CMS) to deliver a seamless and dynamic content experience.

## Technologies Used

This project is built with the following technologies:

- [Next.js](https://nextjs.org): A React framework for production-grade web applications.
- [NextAuth.js](https://next-auth.js.org): Authentication for Next.js applications.
- [Prisma](https://prisma.io): A modern database toolkit for TypeScript and Node.js.
- [Drizzle](https://orm.drizzle.team): A lightweight TypeScript ORM for SQL databases.
- [Tailwind CSS](https://tailwindcss.com): A utility-first CSS framework for rapid UI development.
- [tRPC](https://trpc.io): End-to-end typesafe APIs for TypeScript.
- [Prismic](https://prismic.io): A headless CMS for managing and delivering content.

## Features

- **Dynamic Content Management**: Powered by Prismic CMS for easy content updates.
- **Modern UI/UX**: Built with Tailwind CSS for a responsive and visually appealing design.
- **Authentication**: Secure user authentication using NextAuth.js.
- **Database Management**: Efficient data handling with Prisma and Drizzle ORM.
- **API Integration**: Typesafe APIs with tRPC for seamless data flow.

## Getting Started

To get started with the My Ankle Website, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/Biffnewmedia/my-ankle.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables by creating a `.env` file in the root directory. Refer to the `.env.example` file for required variables.
4. Run the development server:
   ```bash
   npm run dev
   ```
5. Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.
